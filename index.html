How to Play

  1. Paris Square Scene: Use arrow keys to move your character and find the group of tourists.
  2. Introduction Scene: Watch as the vendor shows the ball and prepares the cups.
  3. Game Scene:
    • Watch carefully as the cups are shuffled
    • Notice that another tourist will select one of the cups
    • Click on one of the remaining cups to place your bet
    • Try to keep your money for as long as possible!

Warning: This game simulates a street scam - it's rigged so you'll always lose!

Deploy Your Own Version

Want to host this game yourself? Follow these steps:

Option 1: Using Pygbag (Easiest)

  1. Install pygbag: pip install pygbag
  2. Build the web version: pygbag paris_game.py
  3. Upload the contents of the build/web directory to any web host
